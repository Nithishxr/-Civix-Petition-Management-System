# -Civix-Petition-Management-System
Civix is a full-stack petition management system that enables citizens to create, sign, and track petitions while allowing officials to review and manage them. Built with Node.js, Express, and MongoDB, it features secure JWT authentication, role-based access control, and location-based filtering.
⚙️ Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- REST APIs
  🚀 Features
  - User Authentication (JWT based)
- Role-based system (Citizen & Official)
- Create, edit, and sign petitions
- Location & category-based filtering
- Secure backend API

🛠️ Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Create a `.env` file
4. Run `npx nodemon server.js`
    Environment Variables
   MONGO_URI=
JWT_SECRET=
EMAIL_USER=
EMAIL_PASS=
